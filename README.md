## Features

- Generate professional-looking certificates from multiple online learning platforms.
- Supports custom backgrounds, fonts and text fields for personalization.
- Provides sample templates for Udemy, LinkedIn Learning and more.
- Download the generated certificate as an image or PDF for sharing.

## Installation

This project is a static website built with HTML, CSS and JavaScript. To run it locally:

```bash
# clone the repository
git clone https://github.com/dhruvdaberao/certificate-generator.git
cd certificate-generator

# open index.html directly in your browser
```

Alternatively you can serve the site with a simple HTTP server (optional):

```bash
python -m http.server
```

Then navigate to `http://localhost:8000` in your browser.

## Usage

1. Open the site in your browser.
2. Select the platform (e.g., Udemy, LinkedIn Learning).
3. Enter your name and course details.
4. Preview the generated certificate.
5. Download the certificate as an image or take a screenshot to share.

The site is responsive, but for best results use a desktop browser.
